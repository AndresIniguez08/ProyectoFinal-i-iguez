# Fox Stickers - E-commerce React

Proyecto final desarrollado en React para simular un e-commerce de stickers.

## Tecnologías utilizadas

- React
- React Router
- Context API
- Firebase
- Firestore
- TailwindCSS

## Funcionalidades

- Catálogo de productos
- Filtrado por categorías
- Vista de detalle del producto
- Carrito de compras global con Context
- Checkout con formulario
- Generación de órdenes en Firestore

## Instalación

Clonar repositorio

npm install

npm run dev
